description: 한국어 버전 README로 영어 원본 README를 작성합니다.
---

1. @README.md 를 삭제합니다.
2. @README.ko.md 를 읽고 관련 구현을 분석해서 내용을 이해합니다.
3. 이해한 내용을 기반으로 영어 버전을 작성해서 README.md 로 저장합니다.
   - 한국어 버전에 담긴 의미와 미묘한 정보들이 변하지 않도록 충분히 고민하면서 번역해야 합니다.
   - 영어 버전의 상단 문구는 `English | [한국어](README.ko.md)` 로 작성합니다.
4. 번역 과정에서 README.ko.md 에 담긴 의미가 손실되진 않았는지 검토 합니다.
   - 이슈가 있으면 개선 후 다시 검토합니다. 이슈가 발견되지 않을 때 까지 반복합니다.
